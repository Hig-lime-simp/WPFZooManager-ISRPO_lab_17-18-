НАЗНАЧЕНИЕ:
  1. Простейшая работа с БД, связями, практика написания таких проэктов (Фактическое назначение)
  2. Редактирование БД, добовление зоопарков и животных (Прямое назначение)
ОСНОВНЫЕ ВОЗМОЖСТИ:
"Add Zoo" — добавить новый зоопарк (вводишь название в текстовое поле и жмешь)
"Update Zoo" — изменить название выбранного зоопарка
"Delete Zoo" — удалить выбранный зоопарк (автоматически удаляются и животные из него)

"Add Animal" — добавить новое животное в общий каталог
"Update Animal" — переименовать выбранное животное
"Delete Animal" — удалить животное из каталога

"Add Animal to Zoo" — добавить выбранное животное в выбранный зоопарк
"Remove Animal" — убрать животное из текущего зоопарка

КАК ЗАПУСТИТЬ:
